Redis API com Dart

Para acessar as funções da api é recomendado utilizar o Postman.
Acesse  localhost:8888/redis usando:
  • GET: para buscar todos;
  • /(alguma key) GET: para buscar apenas um;
  • POST {"key": "value"}: para inserir apenas um;
  • /(alguma key) PUT "value": para alterar um valor;
  • /(alguma key) DELETE: para remover uma entrada.
